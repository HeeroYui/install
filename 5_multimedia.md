Multimedia section
==================

Install players
---------------

```
pacman -S vlc
```

Encode DVD
----------

```
pacman -S handbrake libdvdread libdvdcss libdvdnav
```

Transcode TV
------------

```
pacman -S ffmpeg
```


# intal compatibility with new interface of ntfs disk

```
pacman -S ntfs-3g
```
