Multimedia section
==================

Install players
---------------

```
pacman -S vlc
```

Encode DVD
----------

```
pacman -S handbrake libdvdread libdvdcss libdvdnav
```

Transcode TV
------------

```
pacman -S ffmpeg
```


# install compatibility with new interface of ntfs disk

```
pacman -S ntfs-3g
```

# install compatibility with new interface of ex FAT disk

```
pacman -S exfat-utils
```
